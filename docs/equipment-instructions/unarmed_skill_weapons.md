## Format Key
```
Value: Currency cost
Weight: lbs
AP: S = Single, T = Targeted
Damage: Formula + MD (Strength melee bonus)
Ammo Type: Ammo Type if applicable
Range: Hexes/meters
Capacity: Magazine size if applicable
Flavor: Fun Flavor Text
Notes: Mechanical effects only
```
