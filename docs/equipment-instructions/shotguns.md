# Shotguns — Clean Rewrite

## Format Key
```
Value: Currency cost
Weight: lbs
Damage: Ammo Damage × Pellet Count
Ammo Type: Ammo Used
AP: S = Single Shot, T = Targeted Shot
Range: Effective range in hexes/meters
Capacity: Magazine size / reload type
Spread: Pellet pattern behavior
Flavor: Fun flavor text
Notes: Mechanical effects only
Req: Minimum STR (if applicable)
```
