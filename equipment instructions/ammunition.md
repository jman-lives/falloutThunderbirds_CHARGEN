# Ammunition — Clean Rewrite

## Format Key
```
Value: Currency cost (per round unless stated otherwise)
Damage: Base damage added to weapon profile
Armor Mod: Modifier vs. armor
Notes: Mechanical effects only
Special: Unique behaviors
```

```

```
