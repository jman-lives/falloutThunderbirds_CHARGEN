# Primitive Missile Weapons — Clean Rewrite

## Format Key
```
Value: Currency cost
Weight: lbs
Damage: Formula (+ MD when applicable)
AP: S = Standard Attack, T = Targeted Attack
Range: Hexes/meters
Notes: Mechanical effects only
```
