# Machine Guns — Clean Rewrite

## Format Key
```
Value: Currency cost
Weight: lbs
Damage: Ammo Damage + Weapon Bonus (if any)
AP: S = Single Shot, T = Targeted Shot, B = Burst / Full Auto
Range: Effective range in hexes/meters
Capacity: Belt or drum size
Burst: Rounds fired per burst or cycle
Req: Minimum STR (if applicable)
Notes: Mechanical effects only
```
